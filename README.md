# Curso de Desarrollo Web y Metodologías Ágiles

Bienvenido al curso de **Desarrollo Web y Metodologías Ágiles**. En este curso, exploraremos tecnologías fundamentales del desarrollo web y aprenderemos a trabajar con **Scrum** para gestionar proyectos de manera eficiente.

## 📌 Estructura del Curso

El curso está organizado en clases, abordando progresivamente los siguientes temas:

1. **HTML** - Estructura y semántica web.
2. **CSS** - Estilos y diseño responsivo.
3. **JavaScript (JS)** - Programación del lado del cliente.
4. **Express** - Desarrollo de aplicaciones backend con Node.js.
5. **SQL** - Gestión y manipulación de bases de datos relacionales.
6. **React** - Creación de interfaces interactivas y dinámicas.
7. **Scrum** - Metodología ágil para gestión de proyectos.

## 🎯 Objetivos

- Dominar las tecnologías clave del desarrollo web.
- Aprender a construir aplicaciones frontend y backend.
- Comprender los fundamentos de bases de datos con SQL.
- Aplicar la metodología Scrum en proyectos reales.

## 🚀 Requisitos Previos

No se requieren conocimientos previos en programación, pero es recomendable tener nociones básicas de lógica de programación y un gran entusiasmo por aprender.

## 📂 Material y Recursos

Durante el curso proporcionaremos documentación, ejercicios prácticos y proyectos. Además, recomendamos explorar los siguientes recursos:

- [Mozilla Developer Network (MDN)](https://developer.mozilla.org/)
- [Documentación de React](https://react.dev/)
